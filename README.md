## 📌 Sobre o Projeto

O **SmartCondo** é uma plataforma web de gestão condominial desenvolvida para facilitar o dia a dia dos síndicos e moradores. Com uma interface intuitiva, o sistema centraliza todas as demandas do condomínio em um único lugar — desde o controle financeiro até a comunicação com os condôminos.

---

## ✨ Funcionalidades

- 📊 **Dashboard gerencial** com visão geral do condomínio
- 💰 **Gestão financeira** — boletos, inadimplência, receitas e despesas
- 🔧 **Ordens de serviço** — abertura, acompanhamento e encerramento de chamados
- 📢 **Comunicados e avisos** para os moradores
- 🗳️ **Assembleias virtuais** — convocação, votação e ata digital
- 👥 **Cadastro de moradores** e controle de acesso
- 📅 **Reserva de áreas comuns** (salão de festas, churrasqueira, academia etc.)
- 📁 **Repositório de documentos** (regulamento interno, convenção, contratos)
- 📈 **Relatórios e extratos** financeiros e operacionais

## 🚀 Instalação

```bash
# 1. Clone o repositório
git clone https://github.com/igorr0cha/SmartCondo.git

# 2. Acesse a pasta do projeto
cd SmartCondo

# 3. Instale as dependências
npm install

# 4. Configure as variáveis de ambiente
cp .env.example .env
# Edite o arquivo .env com suas configurações

# 5. Execute a aplicação em modo de desenvolvimento
npm run dev
```

A aplicação estará disponível em `http://localhost:3000`.

## 💻 Como Usar

1. Acesse a plataforma e faça login com suas credenciais de síndico.
2. No **Dashboard**, visualize os indicadores gerais do condomínio.
3. Utilize o menu lateral para navegar entre os módulos (financeiro, chamados, comunicados etc.).
4. Cadastre moradores e gerencie permissões de acesso.
5. Acompanhe inadimplência e gere relatórios financeiros.
6. Crie e envie comunicados para todos os condôminos.
